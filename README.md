# TGForwarder

既然你找到了这个项目，我觉得你应该知道他是干什么的

不知道？丢人，退群吧（

---

首先我不建议你分裂生态，开源仅仅是为了安全（虽然我也有可能在服务端使用别的源代码，不过你不信任我也很 不知所措.jpg

而且，代码写的很差，也没有做 Python 版本判断与依赖判断（毕竟最开始的时候是给环境量身打造的

于是这些代码开源的意义仅供参考了

---

```
pip install telethon
python main.py
```
